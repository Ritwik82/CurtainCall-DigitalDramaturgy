# CurtainCall: A Digital Dramaturgy

A theatrical-themed webpage designed to evoke the dramatic flair of a stage play. This project features contrasting colors, elegant typography, and interactive elements to guide the user through different "acts" or sections of the page.

## Features

-   **Dramatic Design:** Uses a dark theme with golden accents for a classic theatrical feel.
-   **Responsive Layout:** Adapts to various screen sizes.
-   **Interactive Scenes:** Navigate through different "acts" with engaging buttons.
-   **Shakespearean Quotes:** Each scene is adorned with famous lines.
-   **Character Spotlights:** Minor interactive elements for character-specific insights.
